## Installation

```bash
npm install -g orbit-cli
```

Or use directly with npx:

```bash
npx orbit-cli create my-app
```

## Usage

### Create a new project

```bash
orbit create my-app
```

### List available frameworks

```bash
orbit list
```

### Check system requirements

```bash
orbit doctor
```
